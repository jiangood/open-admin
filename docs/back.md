```java

```

### `BeanUtils.copy` 示例

```java
class SourceBean {
    String name = "Source";
    Integer value = 1;
}

class TargetBean {
    String name;
    Integer value;
}

SourceBean source = new SourceBean();
TargetBean target = new TargetBean();
BeanUtils.copy(source, target);
// target.name is "Source", target.value is 1
```
